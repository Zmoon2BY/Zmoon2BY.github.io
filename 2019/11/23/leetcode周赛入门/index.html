leetcode周赛入门

发表于 2019-11-23 |

一个非常简单的leetcode周赛入门教程

这几周打了几次leetcode周赛双周赛，感觉题目对新手还是比较友好的，另外leetcode和各大oj的做题界面不太一样，比较像是面试那种白板写代码，比较锻炼once coding的能力，所以哪怕是ACMer or OIer 平时打一打leetcode的比赛也是不错的，而且貌似排名比较靠前的选手也都是现役或者是退役的大神，毕竟比赛有奖金

报名链接 https://leetcode-cn.com/contest/

周赛有两种：

  • 普通周赛 一周一次 周天上午10:30 - 12:00 参加人数相对来说较多
  • 双周赛 两周一次 周六晚上10:30 - 12:00 由于比赛时间原因人数较少

参赛前要进行注册报名比赛，然后到时候参加就可以

以上是leetcode的做题界面，题库里的和竞赛时的做题方式是一样的，所以也可以去题库里多做一些题

因为leetcode的题目是以一个封装好的函数给出来的，函数中的参数不能更改，比如上图的grid，如果你觉得拼写麻烦而把它改成g，那么将无法通过编译

另外输入也是由参数传递进来的，通常为STL如vector，还有int等常规数据类型，如果不熟悉STL的话可以看一下

https://wenku.baidu.com/view/93f33b3b192e45361066f5eb.html STL入门介绍

其实如果是单纯解决输入输出问题的话，知道vector怎么用就可以

比如上图中的grid

做完题目之后“执行代码”用样例进行调试 “提交解答”（字面意思） 值得一提的是答案错误的话leetcode会返回第一个错误的测试用例，可以自定义测试用例之后进行调试然后再提交

leetcode的排名机制比较特殊：

  • 每个题目有对应的分数 按分数高排序（第一关键字）
  • 提交错误有五分钟罚时（如果该题目最后在比赛时间内通过的话，不通过不计）
  • 比赛完成时间以最后一次提交通过的时间 + 罚时 时间长排名靠后（第二关键字）

另外，需要注意！！！

如果报名之后不交题目是算作0题的！！

参与排名！！！

这跟codeforce什么的比赛都不一样！！！

后果：

第一次打完之后全国排名97

一次报名没打 他妈垫底一次 全国排名1w开外了

之后两次打的也不好都没有AK hhh 老年选手暴力写挂了 慢慢来吧用不了几次就回去了

应该是有rating计算系统的，虽然很迷 感觉算的有些问题，rank70只加了200分

注：

  • 报名不打的正确方式是在开始之前取消报名
  • 另外周赛要比双周赛好上分一点，熬夜打比赛的都是大佬！小白多打白天场
